# SiHD Landing Page

Landing estática para GitHub Pages.

## Estructura

- `index.html`: contenido principal.
- `css/style.css`: identidad visual, layout y responsive.
- `js/main.js`: menú móvil, año automático y animaciones suaves.
- `assets/logo.png`: logo.
- `robots.txt` y `sitemap.xml`: SEO básico.

## Publicación

Subir todos los archivos a la raíz del repositorio `sihd-es.github.io`.
